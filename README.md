# 💳 Digital Lending Platform – Credit Risk Analytics System

## 📌 Project Overview
This project is a **Digital Lending Platform Analytics System** designed to predict loan default risk, segment customers, and generate actionable insights for credit decision-making.

It uses machine learning models to assess borrower risk and supports business decisions in digital lending environments.

---

## 🎯 Problem Statement
Financial institutions need to:
- Identify high-risk borrowers early
- Reduce loan defaults
- Improve credit decision accuracy
- Segment customers for better lending strategies

This project solves these challenges using machine learning and analytics.

---

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Matplotlib, Seaborn
- Power BI (or Tableau)
- Jupyter Notebook / Google Colab

---

## 🧠 Machine Learning Models Used
- Logistic Regression
- Random Forest Classifier
- XGBoost Classifier (Best Performing Model)

---

## 📊 Key Features

### 🔹 1. Loan Default Prediction
- Predicts probability of loan default using XGBoost model
- Outputs risk score between 0 and 1

### 🔹 2. Risk Categorization
- Low Risk
- Medium Risk
- High Risk
- Critical Risk

### 🔹 3. Customer Segmentation
- KMeans clustering used to group borrowers into meaningful segments:
  - Prime Borrowers
  - Emerging Borrowers
  - Stressed Borrowers
  - High Risk Borrowers

### 🔹 4. Business Insights
- Helps banks identify risky customers early
- Supports loan approval strategies
- Enables portfolio risk monitoring
